Google Sheets Connect OAuth Example

This application demonstrates how to use Posit Connect's OAuth integration with Google to access the Google Sheets API and list the user's most recent sheets.

This application uses Posit Connect's OAuth integration with Google to access Google Sheets.

List My Recent Google Sheets

How it works:

  1. The application uses Posit Connect's OAuth integration with Google
  2. Connect provides access tokens to securely access Google APIs
  3. The application retrieves and displays your most recent Google Sheets
